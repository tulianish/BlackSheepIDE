# BlacksheepIDE
A java based IDE made for minor project which helps developers

*IDE Features*

- Error Highlighting
 - Syntax Highlighting
 - Basic Grammar Checking
 - Parenthesis Matching
 - Building and Compiling
 - Text Indentation
 - Support for GCC
 - Support .cpp/.cc/.cxx and .h files

*Graphical User Interface Specification*

 - Uses the SWING API
